This cool project idea is a modified version of the concept shown in the video linked at the end of this document. (not my video)

I've done a detailed explanation on how it works, about 2 years ago, in my old blogspot post.

The logic behind the beautiful view created by the app is pretty simple.

Now, here is the basic structure I had in mind while writing this app.

  • Write the droplet class
  • Create a canvas
  • Add thousands of droplets in the canvas
  • Update the position of each droplet every 10 milliseconds

That's it.

Once the app design is finished, writing the code is very easy.

What I did different is that I added some new fields to the droplet class like width and speed so I could give the droplets a depth perception by drawing them as rectangles and not as lines, and set the falling speed accordingly so it appears as if the bigger droplets(those that look like they are in front) are falling faster than those in the back.

If you try this project I'd recommend trying something different, for example add horizontal speed and make the droplets slide a bit to the side to imitate the wind pushing them.

Anyway, this is an old project so keep in mind that it is not as efficient as I would make it now. One of the changes I would make is that I would use Timer class to control the fps of the animation and not the Thread.sleep() since the sleep() method of Thread class is not meant for this kind of things.
