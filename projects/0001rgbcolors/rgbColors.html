This is an Android app made using Android Studio. I wrote this app in Kotlin.

There are 3 sliders by which you control the value of Red, Green and Blue colors. Their values range from 0 to 255, then get used to make the final color which is the combination of the three.

The new color is used to pain the main square.

It was a fun project to make while learning Android App Development
